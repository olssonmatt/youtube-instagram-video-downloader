# Motivation
Project for a CLI vibe coded youtube/instagram video downloader so I can save grappling clips

## Usage Examples
 - Single Short or Reel ``` python main.py download https://youtube.com/shorts/xxxx ```
 - Multiple URLs ``` python main.py download URL1 URL2 URL3 ```
 - YouTube Playlist ``` python main.py download https://youtube.com/playlist?list=XXXX ```
 - Batch File ``` python main.py batch links.txt ```
 - Instagram (with cookies) ``` python main.py download https://www.instagram.com/reel/XXXX --cookies cookies.txt ```
